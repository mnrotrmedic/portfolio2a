#Goals

#Goals is the first 'Full-Stack' app and was a group project. This app was deployed to Heroku (link above) and used Express JS, node, a MySQL database, Sequelize ORM and was styled using Bulma, a CSS library. The team dynamic was great on this project but we learned early on that having too many hands in one Javascript file can lead to trouble. My primary role on this prject was writing the Express routes and related database calls using Sequelize.

Friend Finder

Friend Finder is a 'Full-Stack' app that uses Node JS and Express JS. A user responds to questions which then matches that user based on their similarity in score to other users. It's pretty basic and the questions came directly from the assignment. It was great building an app from the ground up that was deployable as a 'real' website though.

Train Scheduler

The Train Scheduler app higlights the use of moment.js to make working with time data a bit easier. It also uses Google Firebase as a modular back-end service to house the database information. Firebase stores the 'train' information as it is entered while moment handles the math for everything else on the page.
